# Data Visualization and Business Insights

## Project Overview
This project focuses on creating interactive visualizations and generating business insights from sales data using Plotly.

## Objective
- Visualize sales performance.
- Identify top-performing products.
- Analyze sales trends over time.
- Understand customer payment preferences.
- Generate actionable business insights.

## Tools Used
- Python
- Pandas
- Plotly Express
- Jupyter Notebook

## Visualizations Created

### Revenue by Product
- Bar Chart
- Identifies highest revenue-generating products.

### Sales Trend Over Time
- Line Chart
- Shows sales growth and peak sales periods.

### Payment Method Distribution
- Pie Chart
- Displays customer payment preferences.

## Key Insights
- Identified the top revenue-generating product.
- Determined peak sales periods.
- Analyzed customer payment behavior.
- Converted raw sales data into meaningful visual stories.

## Output
Interactive dashboards and visual reports that support data-driven decision-making.
